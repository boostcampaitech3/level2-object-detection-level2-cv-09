## Overview

제출한 submission.csv를 다시 **COCO dataset**으로 변환합니다.  

## process
1. 변환할 submission.csv 파일을 현재 경로로 가져옵니다.
2. submission2json.jpynb 실행시 **COCO dataset** format으로 현재 경로에 submission_coco.json으로 생성됩니다.
