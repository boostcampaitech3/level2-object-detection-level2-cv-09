### Stratified K-fold (by JJ)

- 파이썬 라이브러리 설치
```
pip install funcy
```

- 사용 방법
```
python stratified_kfold_group.py -n [Number of Splits (default=5)] [Data Directory]
```


- 사용 예시
```
python stratified_kfold_group.py -n 5 ../../detection/dataset/
```
